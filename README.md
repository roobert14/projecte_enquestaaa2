Es un projecte web senzill per recollir i visualitzar respostes d'una enquesta d'aula. L'aplicacio esta feta en un unic fitxer `index.html`, amb HTML, CSS i JavaScript.

## Com obrir-lo en local

1. Descarrega o obre aquesta carpeta al teu ordinador.
2. Fes doble clic a `index.html` per obrir-lo amb el navegador.

Tambe pots obrir-lo des de l'editor amb l'opcio de previsualitzacio o amb una extensio com Live Server.
